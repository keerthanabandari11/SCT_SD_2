# Number Guessing Game

## Description
This Python program generates a random number and asks the user to guess it.

## Features
- Random number generation
- User guessing system
- Hint messages:
  - Too High
  - Too Low
- Loop until correct guess

## Technologies Used
- Python
- Random Module

## How to Run

```bash
python guess_game.py
Example Output
